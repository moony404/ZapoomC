
# ZapoomC

## Run
How to run my project ?
- With gcc:
```
  gcc [name_of_the_source_code].c -o [name_of_the_exe_file].exe
```
- With the exe file :
```
Just open the exe file and accept all of the Windows warning pop-ups
```
